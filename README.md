# Superstore Sales Analysis

This project analyzes Superstore sales data using SQL and visualizes insights with Tableau.

## Tools

* MySQL
* SQL
* Tableau

## Dataset

Superstore Sales Dataset

## Analysis

* Sales by category
* Sales by region
* Top customers
* Shipping mode analysis

## Dashboard

A Tableau dashboard was created to visualize key insights from the dataset.
